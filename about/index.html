※ NavBar 기능과 사용

   

1. Home  

     - 블로그의 메인화면으로 이동합니다.

     - 사용자가 지정한 블로그의 Feature Post가 출력됩니다.


2. About  

     - 블로그의 도움말 페이지로 이동합니다.


3. Category Post  

     - 카테고리 별 포스트를 출력하는 화면으로 이동합니다.

     - 카테고리 별로 포스트이 작성날짜에 따라 배열됩니다.


4. All post  

     - 모든 포스트를 출력합니다.

     - 작성날짜에 따라 포스트들이 배열됩니다.


5. Tag 별 post  

     - 태그(주제)별 도움말 페이지로 이동합니다.

     - 주제별로 작성된 포스트들의 대표내용이 모두 출력됩니다.





※ 편집자 페이지

   
- 편집자 페이지는 편집자의 정보를 출력하기 위한 페이지입니다. - 출력하는 편집자의 정보는 아래와 같습니다.

 -> 이름, 닉네임, 지위, 자기소개, 이메일, 소속, ID, 가입일, 소셜





※ 검색상자

   
- 특정 포스트를 검색하기 위하여 사용합니다.
